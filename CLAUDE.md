# Portable Brain

This folder contains a Portable Brain — a personal knowledge system powered by AI.

## Session Start
Check if `BOOTSTRAP.md` exists in this directory.
- **If it exists:** This is a first-time setup. Follow the instructions in BOOTSTRAP.md instead of anything below.
- **If it doesn't exist:** The vault is initialized. Follow the Session Start Protocol in `06-Agent/workspace/AGENTS.md`.

## Vault Map
| Folder | Purpose |
|--------|---------|
| `00-Inbox/` | Capture — everything lands here first |
| `01-Projects/` | Active work with a finish line |
| `02-Areas/` | Ongoing responsibilities |
| `03-Resources/` | Knowledge base — evergreen reference |
| `04-Archive/` | Completed or abandoned work |
| `05-Meta/` | How this vault works |
| `06-Agent/` | AI agent runtime, memory, automation |
| `07-Systems/` | CRM, finances, planning |
| `08-CoreSystem/` | Personal operating system |
