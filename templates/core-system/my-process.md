How {{USER_NAME}} works. How decisions get made. The operating rhythm.

## Overview

The process rests on three parts: **Capture · Process · Review**.

### 1. Capture
One inbox for everything. Ideas, todos, commitments captured during the day,
meetings, or conversations. Don't file in real-time — capture first, file later.

- Primary capture: `00-Inbox/`
- Secondary capture: daily notes (for things that come up mid-meeting)

### 2. Process
Dedicated windows to triage what was captured.

**Planning Touchpoints:**
- **Annual** — define the year's big challenge and 2-3 themes
- **Quarterly** — set Big Rocks (2-3), define focus areas
- **Weekly** — Friday review: score this week, set next week's Big 3
- **Daily** — morning briefing, evening close-out

### 3. Review
Regular looks back:
- End of day: memory log, open loops, tomorrow's priority
- End of week: Friday review, score Big 3, surface carries
- End of quarter: Rock assessment, adjust for next quarter
- End of year: retrospective, set next year's challenge
